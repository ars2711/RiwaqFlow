Accessibility & trust

Built to a floor, not just a look

✓ Contrast

Body text targets 4.5:1 against its background in both themes; large display type and icons target 3:1. Muted text was darkened specifically to clear this on the paper background.

✓ Keyboard & focus

Every interactive element — the theme toggle, badges intended as links, ticket actions — gets a visible 2px focus ring in the verified-green, never relies on hover alone.

✓ Motion

The old glowing pulse ring is gone. Status is shown as a static dot with a text label — calmer, and nothing here moves unless it's telling you something changed.

✓ Colour-blind safe

Verified and alert states are never colour-only — each pairs a checkmark or warning icon with an explicit word, so the meaning survives grayscale.

On the security signal specifically: the green seal and rotating ticket number are designed to mirror what the backend already does in the real Riwaq build — signed, time-rotating tokens locked to one device. The visual trust mark exists to represent a real guarantee, not to stand in for one that isn't built yet.
